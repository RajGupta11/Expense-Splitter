# Expense-Splitter
# 💸 Smart Expense Splitter

## 📌 Description

Smart Expense Splitter is a Java-based desktop application that helps users easily manage and split shared expenses among friends, roommates, or teams.

It simplifies tracking of who paid, who owes whom, and automatically calculates balances and settlements.

---

## 🚀 Features

* 👥 Add multiple users dynamically
* 💸 Add expenses with description and amount
* 🧾 Select **“Paid By”** user
* ⚖️ Automatic equal expense splitting
* 📊 Real-time balance calculation
* 🔄 Smart settlement (who pays whom)
* 🧠 Basic AI-like expense categorization
* 🖥️ Clean and interactive Java Swing UI

---

## 🛠️ Tech Stack

* **Language:** Java
* **UI:** Swing
* **Concepts:** OOP (Object-Oriented Programming)

---

## ▶️ How to Run

### 🔧 Compile

```
javac model/*.java service/*.java ui/*.java Main.java
```

### ▶️ Run

```
java Main
```

---

## 🧪 Example Usage

### Step 1: Add Users

```
Raj
Aman
Priya
```

### Step 2: Add Expense

```
Description: Pizza  
Amount: 900  
Paid By: Raj
```

### Output:

```
Aman owes Raj Rs.300
Priya owes Raj Rs.300
```
